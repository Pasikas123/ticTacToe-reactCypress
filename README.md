### Installing

1. Open Visual Studio Code.
2. Clone this project:
   ```
   git clone https://github.com/ddailide/ticTacToe-reactCypress.git
   ```
3. Run in terminal:
   ```
   npm install
   ```
4. Run in terminal:
   ```
   npm run dev
   ```
5. Open link:
   ```
   http://localhost:5173/
   ```
